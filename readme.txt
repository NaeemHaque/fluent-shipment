A comprehensive shipment tracking and management solution for WordPress with support for multiple e-commerce platforms.

== Description ==

Fluent Shipment is a powerful WordPress plugin designed to streamline shipment tracking and management for e-commerce websites. Built with modern PHP and Vue.js, it provides a complete solution for managing deliveries, tracking packages, and coordinating with delivery riders.

**Key Features:**

* **Multi-Platform Support**: Works with WooCommerce, Fluent Cart, and supports manual/API shipment creation
* **Real-Time Tracking**: Complete shipment tracking with multiple status updates
* **Rider Management**: Comprehensive rider assignment and management system
* **Dashboard Analytics**: Beautiful dashboard with charts and statistics powered by ECharts
* **Email Notifications**: Automated customer notifications for shipment updates
* **Advanced Search & Filtering**: Filter shipments by status, date range, tracking number, and more
* **Modern UI**: Clean, responsive interface built with Vue.js and Element Plus
* **RESTful API**: Complete API for third-party integrations
* **Tracking Events**: Detailed event logging for each shipment stage

**Shipment Statuses:**
* Pending
* Processing
* Shipped
* In Transit
* Out for Delivery
* Delivered
* Failed
* Cancelled

**Perfect for:**
* E-commerce stores using WooCommerce
* Fluent Cart users
* Logistics companies
* Delivery service providers
* Any business requiring shipment tracking

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/fluent-shipment` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Run `composer install` in the plugin directory to install dependencies.
4. Run `npm install && npm run build` to build frontend assets.
5. Use the Fluent Shipment menu to configure your settings.

== Frequently Asked Questions ==

= Does this plugin work with FluentCart? =

Yes, Fluent Shipment has built-in support for FluentCart integration.

= Can I track shipments in real-time? =

Yes, the plugin provides real-time tracking with multiple status updates and event logging.

= Does it support email notifications? =

Yes, automated email notifications are sent to customers for important shipment updates.

= Can I assign riders to deliveries? =

Yes, the plugin includes a comprehensive rider management system with assignment capabilities.
